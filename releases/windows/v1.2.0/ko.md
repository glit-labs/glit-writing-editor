## 새로운 기능 / 개선 사항

### 테마 개편 — 기본(System)·종이(Paper) 추가, 클린 제거

테마 구성이 개편되었습니다. 기본(System)과 종이(Paper) 테마가 추가되고, 클린(Clean) 테마는 폐지되었습니다.

- **기본(System)**: 운영체제의 라이트/다크 모드를 자동으로 따라가는 중성 팔레트입니다. 시스템이 다크 모드로 전환되면 본문과 타이틀바가 즉시 함께 바뀝니다.
- **종이(Paper)**: 밝은 종이 느낌의 배경에 차분한 남색 강조색을 사용하는 테마입니다.
- 기존에 클린 테마를 사용하던 경우 자동으로 종이 테마로 전환됩니다.
- 신규 설치 기본 테마는 기존과 동일하게 온기(Warmth)입니다.

### 의도치 않은 개행 식별 개선 (한국어)

입력 중 자동 줄바꿈과 실제 개행(Enter)을 구분하기 어려운 문제가 개선되었습니다.

- **개행 표시** 토글: 실제 개행 위치에 옅은 `↵` 마커를 표시합니다. 본문 줄바꿈 위치에는 영향을 주지 않으며 기본은 꺼짐입니다.
- **줄바꿈 방식** 선택(한국어): 음절 단위(기본)와 어절 단위 중 선택할 수 있습니다. 음절 단위는 웹소설 리더와 동일한 방식으로 본문을 표시합니다.

### 미리보기(디바이스) — 설정을 옵션 팝업으로 이동

미리보기(디바이스) 창 툴바의 디바이스 선택과 글자 크기 조절이 "설정" 팝업으로 정리되었습니다. 창 제목은 가운데로 정렬되었습니다.

### 이미지 뷰어 UI 정리

이미지 뷰어의 파일명이 창 가운데로 정렬되고, 줌 컨트롤이 하단 상태바 오른쪽으로 이동했습니다.
