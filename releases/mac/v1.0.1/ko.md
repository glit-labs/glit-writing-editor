# Glit 릴리즈 노트 — v1.0.1

**출시일:** 2026년 5월  
**이전 버전:** v1.0.0

---

## 새로운 기능

### 오른쪽 보조 사이드바 (인라인 미리보기 패널)

툴바 우측에 보조 사이드바 토글 버튼이 추가되었습니다. 버튼을 누르면 앱 오른쪽에 인라인 미리보기 패널이 열립니다.

- 마크다운 파일은 기존 미리보기와 동일한 렌더링 및 보안 처리 적용 (외부 이미지 차단, 링크는 시스템 브라우저로만 열기)
- 패널 너비를 드래그로 조절할 수 있으며, 설정이 자동 저장됩니다.
- 저장 상태 표시("저장 중" / "저장됨")가 툴바 우측 버튼에서 에디터 상단 우측 텍스트로 이동하여 더 직관적으로 확인할 수 있습니다.

### 사이드바 항목 선택 하이라이트

사이드바에서 파일 또는 폴더를 클릭하면 **선택 상태(테두리)**가 표시됩니다.

- 현재 편집 중인 파일(주황 배경)과 선택된 항목(테두리)을 동시에 구분할 수 있습니다.
- 단순 선택만 하고 에디터 포커스는 유지됩니다.

### 우클릭 컨텍스트 메뉴 및 복사·붙여넣기 단축키

사이드바 항목을 우클릭하면 컨텍스트 메뉴가 나타납니다.

| 항목 | 설명 |
|------|------|
| 새 문서 만들기 | 선택한 폴더 안에 새 파일 생성 |
| 새 폴더 만들기 | 선택한 폴더 안에 새 폴더 생성 |
| Finder에 표기 | OS 탐색기에서 해당 항목 위치 열기 |
| 복사 | 폴더 또는 파일을 클립보드에 복사 |
| 붙여넣기 | 복사한 항목을 현재 폴더에 붙여넣기 (이름 충돌 시 자동 접미사 처리) |
| 이름 바꾸기 | 항목 이름 편집 |
| 삭제 | 항목 삭제 |

키보드 단축키: `Cmd+C` / `Cmd+V`

> 사이드바 항목을 클릭하면 aside 영역으로 포커스가 전환되어 에디터 붙여넣기와 충돌하지 않습니다.

### UI 텍스트 크기 설정 전체 적용

설정에서 지정한 UI 텍스트 크기가 사이드바, 에디터, 검색, 툴바, 설정 패널 등 **모든 컴포넌트**에 일관되게 반영됩니다. 이전 버전에서는 일부 영역에 하드코딩된 크기가 남아 있었습니다.

### 세계관 폴더 파일 이동·정렬 개선 및 이미지 배지

- 세계관 하위 항목을 드래그할 때 파일시스템 이동이 올바르게 동작합니다.
- 사이드바에서 이미지 파일(`.png`, `.jpg` 등)에 확장자 배지가 표시됩니다.
- 세계관 폴더 내 항목의 순서 변경(드래그 재정렬)이 정상적으로 작동합니다.

### 오른쪽 보조 사이드바 — 파일 정보 패널

오른쪽 보조 사이드바에 **파일 정보** 토글이 추가되었습니다. 현재 열려 있는 파일의 상세 메타데이터를 확인할 수 있습니다.

- 표시 항목: 파일 이름, 워크스페이스, 형식, 크기, 글자 수(공백 제외/포함), 최초 생성 시각, 수정 시각, 경로
- 파일을 저장할 때마다 크기가 자동으로 갱신됩니다.
- 웹소설 집필 모드의 **완성 원고 일괄 보기** 중에는 패널이 자동으로 집계 통계 화면으로 전환되어 에피소드 수와 전체 글자 수를 보여줍니다. "자세히" 버튼으로 에피소드별 글자 수 목록을 펼쳐볼 수 있습니다.

### 편집 옵션 패널 — 교정 기능 (웹소설 집필 모드)

웹소설 집필 모드 활성 시 편집 옵션 패널에 세 가지 교정 섹션이 추가됩니다.

**따옴표 교정**

| 항목 | 설명 |
|------|------|
| 둥근 → 곧은 | 둥근 따옴표를 직선 ASCII 따옴표로 변환 |
| 곧은 → 둥근 | 직선 ASCII 따옴표를 둥근 따옴표로 변환 |
| 대사 스타일 정리 | 대사 형식을 통일 |

**마침표 교정** _(한국어 전용)_

| 항목 | 설명 |
|------|------|
| 마침표 → 가운뎃점 | `...` 줄임표를 `···` 가운뎃점 스타일로 변환 |

**들여쓰기 교정**

| 항목 | 설명 |
|------|------|
| 문단 들여쓰기 정리 | 각 문단 앞 공백을 공백 1칸으로 통일 |

교정이 실행되는 동안 에디터 위에 비활성 오버레이가 표시되어 실수로 내용이 변경되는 것을 방지합니다. 각 버튼은 처리 중 진행 표시(`···`)를 보여주고, 완료 후 잠깐 강조 색상으로 피드백을 줍니다.

## 버그 수정

### 파일 이름 변경 후 편집기 상태 미갱신 → 잘못된 경로에 저장되는 버그

파일이 편집기에 열린 상태에서 사이드바에서 이름을 바꾸면, `currentFile.path`와 `draftBuffer` 키가 이전 경로로 남아 있어 저장 시 **삭제된 옛 경로로 파일이 새로 생성**되는 버그가 수정되었습니다.

### 에피소드 읽기 전용 상태가 즉시 반영되지 않는 버그

범위 폴더에서 에피소드 상태를 "완성"으로 변경했을 때, 열려 있는 편집기의 읽기 전용 상태가 즉시 전환되지 않던 문제가 수정되었습니다.

### 워크스페이스 닫을 때 텍스트 검색 결과 미초기화 버그

워크스페이스를 닫아도 이전 검색 결과가 화면에 남아 있던 버그가 수정되었습니다.

### 다국어 초기 로드 플래시 및 언어 변경 버그

- 앱 시작 시 언어가 잠깐 깜빡이던(flash) 현상이 제거되었습니다. preload 단계에서 언어를 동기적으로 미리 읽어 초기 렌더링부터 올바른 언어가 표시됩니다.
- 설정에서 언어를 변경해도 일부 UI가 이전 언어로 표시되던 버그가 수정되었습니다.
- 설정 언어 목록 드롭다운에 일부 언어가 나타나지 않던 문제가 수정되었습니다.

### 범위 폴더 재편성 시 앞쪽 이탈 에피소드 삭제 버그

범위를 축소할 때 범위 앞쪽에서 이탈하는 에피소드가 목록에서 삭제되던 버그가 수정되었습니다. 또한 Google Drive 경로에서 한글 파일명 NFD → NFC 정규화 누락으로 발생하던 재편성 오류도 함께 수정되었습니다.

### 사이드바 드래그 이동 범위 제한

초안·완성 원고 폴더로 외부 항목(다른 폴더의 파일)을 드래그해 이동하는 것이 차단됩니다. 잘못된 드래그로 인한 파일 위치 오염을 방지합니다.

### 따옴표 교정 예시 문자 오류

편집 옵션 패널(웹소설 집필 모드)의 따옴표 교정 섹션에서 **둥근 → 곧은**과 **곧은 → 둥근** 두 항목의 예시 텍스트가 시각적으로 구분이 불가능했습니다. 이제 각 항목의 변환 결과 쪽에 직선 ASCII 따옴표와 둥근 따옴표가 올바르게 표시됩니다.

## 개선 사항

- 읽기 전용 편집기 상단 바에 **글자 수**가 추가되고 레이아웃이 개선되었습니다 ("읽기 전용" 중앙 고정, 글자 수·미리보기 버튼 우측 정렬).
- 완성 원고 일괄 보기 상단에도 **총 글자 수**가 표시됩니다.
- 드래그 이동 후 `draftBuffer` 내부 경로가 올바르게 갱신됩니다.
