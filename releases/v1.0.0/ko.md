# Glit 릴리즈 노트 — v1.0.0

**출시일:** 2026년 4월  
**최초 공개 출시**

---

## 개요

Glit은 장편 소설, 특히 웹소설 집필을 위한 데스크톱 글쓰기 앱입니다. Electron 기반으로 macOS와 Windows에서 네이티브로 실행되며, 모든 파일은 로컬 파일시스템에 직접 저장됩니다.

---

## 핵심 기능

### 웹소설 집필 모드

연재 소설을 체계적인 폴더 구조로 관리하는 전용 모드입니다.

- **초안 폴더**에는 작성 중인 에피소드, **완성 원고 폴더**에는 완료된 에피소드를 보관합니다.
- **범위 폴더**는 에피소드를 회차 단위(예: 1~50화, 51~100화)로 자동 그룹화합니다. 에피소드 생성 시 해당 범위 폴더에 자동 배치됩니다.
- **완성 원고 일괄 보기**에서 완성된 에피소드를 한 화면에서 스크롤하며 확인할 수 있고, 에피소드별·전체 글자 수도 표시됩니다.
- 에피소드 상태(진행 중 / 완성)를 전환할 수 있으며, 완성 에피소드는 읽기 전용으로 열립니다.
- **세계관 폴더**에는 인물 프로필, 배경 설정 등 참고 자료를 원고와 함께 보관할 수 있습니다.

### 멀티 워크스페이스

여러 워크스페이스를 동시에 열 수 있습니다. 각 워크스페이스는 디스크상의 폴더에 대응하며 사이드바에 별도 항목으로 표시됩니다.

### 편집기

- 글꼴 크기·줄간격 설정이 가능한 플레인 텍스트 편집기
- 콘텐츠 길이에 따라 자동으로 높이가 늘어나는 textarea (고정 뷰포트 없음)
- 일치 항목 강조 및 결과 목록을 지원하는 **찾기·바꾸기** 바
- 파일 닫기 또는 워크스페이스 전환 시 미저장 변경 사항 확인 팝업

### 멀티 윈도우

파일을 별도 플로팅 창에서 열 수 있습니다. 메인 창에서 글을 쓰면서 참고 자료를 다른 창에 띄워두는 데 유용합니다.

### 이미지 인라인 미리보기

이미지 파일(`.png`, `.jpg`, `.gif`, `.webp` 등)을 편집기 영역에서 인라인으로 바로 미리볼 수 있습니다. − / + 버튼 또는 `Cmd/Ctrl+스크롤`로 확대·축소할 수 있습니다.

### 마크다운 미리보기 창

`.md` 파일을 독립 미리보기 창에서 렌더링해 볼 수 있습니다. 외부 이미지 차단, 링크는 시스템 브라우저로만 열기 등 보안 샌드박스가 적용됩니다.

### 다국어 지원

UI가 **8개 언어**로 완전히 현지화되어 있습니다: 한국어, English, 日本語, 简体中文, Deutsch, Français, Español, Italiano. 최초 실행 시 OS 언어를 자동 감지하며, 설정에서 언어를 변경할 수 있습니다.

### 테마

**라이트**, **다크**, **세피아**, **다크 세피아** 4가지 내장 테마. 메인 창과 보조 창 전체에 일관되게 적용됩니다.

### 설정

- **편집기** — 글꼴 크기, 줄간격, 자동 저장 토글
- **외관** — 테마 선택, UI 텍스트 크기

### macOS 및 Windows 지원

macOS(Mac App Store)와 Windows(Microsoft Store) 네이티브 빌드를 제공합니다. 각 플랫폼의 창 스타일과 키보드 규칙을 따릅니다.
