## 当方によるデータ収集はありません
当方は自社サーバーを運用せず、いかなる個人情報も収集・保存・共有しません。Glit には解析・広告・追跡のソフトウェアは含まれていません。アプリの利用にアカウントや登録は不要です。

## ファイルはお使いのデバイスに保存されます
あなたが作成または編集したすべてのプロジェクトとファイルは、お使いのデバイスにローカル保存されます。

## クラウドストレージ（任意）
Glit はクラウドストレージを必要とせず、完全にオフラインで動作します。ご利用になる場合、その仕組みはプラットフォームによって異なります:

**macOS と Windows の場合:** クラウド提供元自身のデスクトップアプリ（Google Drive、Dropbox、OneDrive など）をインストールすると、コンピューター上に同期フォルダが作成され、その中で作業を同期します。Glit は他のローカルフォルダと同じように、そのフォルダでファイルを開いて保存するだけです。Glit 自体がクラウドサービスに接続したり、あなたのアカウントにアクセスしたりすることはありません。

**iPhone と iPad の場合:** Glit は公式 API を通じて Google Drive、Dropbox、Microsoft OneDrive に直接接続し、あなたが選んだファイルを同期します。サインインはサービスの安全な OAuth フローで行われ、認証トークンはお使いのデバイスに安全に保存され、当方に送信されることはありません。Glit は作業の同期に必要なファイルにのみアクセスし、アカウントのパスワードには決してアクセスしません。

## サードパーティのサービス
（どのプラットフォームでも）クラウドサービスを利用する際、そのサービスは各社独自のプライバシーポリシーに従います:

- Google（Google サインイン、Google Drive）: <https://policies.google.com/privacy>
- Dropbox: <https://www.dropbox.com/privacy>
- Microsoft（OneDrive）: <https://privacy.microsoft.com/privacystatement>

## 子どものプライバシー
Glit は子どもを対象としておらず、誰からも故意に個人情報を収集することはありません。
